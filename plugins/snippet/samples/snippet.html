Default highlighting highlight.js

Highlighting with client-side default highlighter, using highlight.js library.

Select template: 

CKEDITOR.replace( 'editor1', {
	plugins: 'wysiwygarea,clipboard,toolbar,sourcearea,htmlwriter,basicstyles,enterkey,snippet',
	height: 440
} );

Server-side highlighter

You can extend highlighting functinallity by creating your own plugin which will take care of syntax highlighting. In this case we will use snippetgeshi plugin in order to use GeSHi as highlight provider.

In this case we will also use snippet_langs config variable, to reduce available languages list.

CKEDITOR.replace( 'editor2', {
	plugins: 'wysiwygarea,clipboard,toolbar,sourcearea,htmlwriter,basicstyles,enterkey,snippet,snippetgeshi',
	height: 440,
	snippet_langs: {
		html4strict: 'HTML',
		html5: 'HTML5',
		javascript: 'JavaScript',
		php: 'PHP',
		sql: 'SQL'
	}
} );
