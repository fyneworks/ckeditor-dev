CKEditor Samples » Media Embed

Media Embed plugin allows you to insert videos, images, tweets, music and much more from popular services.
Simply enable plugin and paste link to desired content. You can also use toolbar button to open a dialog where you can insert the link. 

CKEDITOR.replace( 'textarea_id', {
	extraPlugins: 'mediaembed'
} );
